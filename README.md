## Quick Start

### Installation

Create a new React Native App:

```bash
$ react-native init HelloWorld
$ cd HelloWorld
```

Install `@ecreeth/rn-flex` to your project:

```bash
$ yarn add @ecreeth/rn-flex or
$ npm install @ecreeth/rn-flex
```
